<a name="项目介绍介绍"></a>  
## 项目介绍  
  奇迹才聘网，是一个专注于互联网业人才招聘应聘的网站。为求职者提供招聘信息，为公司人事部门提供应聘信息，提高公司工作效率，在求职者和公司之间建立信息联系，方便求职者了解招聘信息，同时也能让公司找到满意的人才。
